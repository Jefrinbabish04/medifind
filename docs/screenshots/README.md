# Screenshot guide for GitHub README

Take these screenshots after running the app locally (`python manage.py runserver`).

Save each file in this folder with the exact name below so the main README shows images correctly.

| File name | What to capture |
|-----------|-----------------|
| `01-customer-home.png` | Customer home page with search box |
| `02-search-results.png` | Search results for "Dolo 650" |
| `03-enquiry-form.png` | "I Need This" enquiry form |
| `04-customer-dashboard.png` | Customer dashboard with enquiry status |
| `05-shop-login.png` | Shopkeeper login page |
| `06-shop-dashboard.png` | Shopkeeper dashboard with charts |
| `07-shop-inventory.png` | Inventory list (stock numbers visible) |
| `08-shop-enquiries.png` | Enquiries table with Accept/Reject buttons |
| `09-demo-flow.png` | Optional: side-by-side or collage of the full demo |

**Tip:** Use Windows Snipping Tool or `Win + Shift + S`, then save PNG files here.
